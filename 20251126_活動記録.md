# 課題の模範解答からクローンを作成
- GitHubを開く。
- 右上の自身アカウント名にある Switch dashboard context から dse-leading-edge-technology-study を選択する。
- 左メニューから Browse organization's repositories を押下する。
- learn-vr を押下する。
- 右上の CODE を押下する。
  ![image001](/capture/20251126_001.jpg)
- Open with Visual Studio を押下する。
  ![image002](/capture/20251126_002.jpg)
- フォルダを指定しクローンを作成する。
  ![image003](/capture/20251126_003.jpg)
- Unity Hub を開き、Add → Add project from disk で上で指定してフォルダを選択する。
  ![image004](/capture/20251126_004.jpg)
- Visual Studioでソリューションファイルを開く。
  ![image005](/capture/20251126_005.jpg)
- Visual Studio右下から ブランチ → リモート → subject/goal を選択する。
  ![image006](/capture/20251126_006.jpg)
- Discard を選択し checkoutを続行するを押下する。
  ![image007](/capture/20251126_007.jpg)
- Projectウィンドウから Assets → _Project → Scenes → Goal → Goal をダブルクリックする。
  ![image008](/capture/20251126_008.jpg)
- Play 押下で動作確認できる。  
  　　参考：キャノンの壁リスポーン方法。  
  　　　　Hierarchyウィンドウから Goal → Cannon → Target を選択し、  
  　　　　Inspector → Target Organizer の...から Spown Targets を押下する。
  ![image009](/capture/20251126_009.jpg)

# ブランチを切る
- 新しいブランチを押下し、ブランチ名を記入後、作成を押下する。

# シーンを作成する
- Projectウィンドウから Assets → _Project → Scenes - Team_C を右クリックし、  
  Create → Scene → Scene で新しいシーンを作成する。

# ドキュメントの保存方法
- Visual Studio のプロジェクトを右クリックし、追加 → 新しいフォルダでDocを作成する。  
  作成したフォルダにテキスト等を保存する。
  ![image013](/capture/20251126_013.jpg)

# 課題内容
１.特定の床に乗ると、物が落ちてくる。  
２.自動ドア(乗ると開く。降りると閉まる。)  
３.手動ドア(ボタン押下で開く。開いた後、数秒後で閉まる。開閉時、ボタンの色が変わる。)  
  　　ボタンクリックは、TとShift + T
